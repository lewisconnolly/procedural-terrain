# procedural-terrain
A simple game created in a DirectX 11 framework, featuring procedural terrain generation using the Midpoint Displacement/Diamond-square algorithm.

Before running, download and extract [textures](https://drive.google.com/file/d/163sip4UH2kNH8MKiYmhmRBVeJBMgUM0N/view?usp=sharing) to root folder.

| Control      | Action                                 |
| ------------ | -------------------------------------- |
| Mouse        | Camera rotation                        |
| WASD			   | Movement                               |
| Left Click   | Shoot                                  |
| Left Ctrl		 | Release cursor                         |
| Right-click  | Re-focus cursor                        |
| M				     | Generate midpoint displacement terrain |
| Space			   | Generate sine and cosine terrain       |
| Shift (hold) | Increase movement speed                |
| F				     | Smooth terrain                         |
| C				     | Respawn collectables                   |
| Alt + Enter	 | Fullscreen                             |
| Esc				   | Quit                                   |
